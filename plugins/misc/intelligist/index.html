Sharing code is an essential part of the Internet. It's how we learn.
The Gist service that GitHub provides is an instrumental part of that sharing process,
making it easy to embed code into your website, and collaborate.

Intelligist is a convenient plugin for sharing GitHub gists on your website.
It allows you to share one or more gists, and even execute the code, which makes it ideal for demoing JavaScript and CSS in-page.

Here it is. You'll notice that, for the most part, it looks like a normal gist, except for a drop-down menu.
