glideTo

glideTo helps you to create modern scrolling websites (like this one) easily

Right now it is being used at Out of the Box, an interactive agency on Mexico City

Author

The plugin is written on CoffeeScript by Jeduan Cornejo

Special Thanks

The smooth scrolling bit was lifted from CSS Tricks

I got great help on the API design from Daniel Zavala

Download

$ git clone git://github.com/jeduan/jquery-glideto
You can also download this project in either zip or tar formats.
